🧶 Loja de Crochê - Little Bunny's- Telas Figma
Este repositório contém a interface desenvolvido no Figma para uma loja de crochê. O objetivo do projeto é oferecer uma navegação simples, intuitiva e agradável para os clientes que desejam explorar e adquirir produtos artesanais.

🎨 Telas Desenvolvidas
Tela Inicial
Página principal da loja, onde os usuários podem visualizar destaques, categorias e produtos em destaque.

Tela de Carrinho de Compras
Permite aos usuários visualizar os itens adicionados, ajustar quantidades e seguir para o pagamento.

Tela de Perfil
Área do usuário com informações pessoais, histórico de pedidos e opções de edição de dados.

Tela de Pagamento
Interface para finalização da compra, escolha de métodos de pagamento e confirmação do pedido.

Tela de Login/Cadastro
Tela para que novos usuários possam se cadastrar ou usuários existentes possam realizar o login na plataforma.

🚀 Tecnologias Utilizadas
Figma (Design de Interfaces e Prototipagem)
